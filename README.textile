<H1>NAME</H1>

<P>
watchng - runs a command periodically, showing output only when it changes.
</P>

<H1>SYNOPSIS</H1>

<P>
**watchng** -h
</P>
<P>
**watchng** [options] &lt;command&gt;
</P>

<H1>DESCRIPTION</H1>

<P>
**watchng** is a program that runs the given command-line periodically, showing
the output only when it is different than the last, along with the time.
</P>

<H1>OPTIONS</H1>

<P>
**-h**
        Help.
</P>
<P>
**-p, --period=&lt;n&gt;**
        Use &lt;n&gt; as period between re-runs. Default is 1s.
</P>
<P>
**-s, --shell**
        Run args with /bin/sh -c
</P>
